We need to prove $P(X\le x)$ = $F(x)$

 $P(X\le x)$ = $P(F^{-1}(U) \le x)$

 = $P(U \le F(x))$ (F is monotone increasing function, $F(F^{-1}(U))=U$)

 = $F(x)$  (U is uniform on [0,1], so $P(U \le F(x))$ = $F(x)$ )

 Therefore, cdf of x is F
