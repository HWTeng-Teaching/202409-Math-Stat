### Prove: If A ∈ B, P(A) ≤ P(B)
## Answer
Since A ∈ B, we can know that every outcome in A is also outcome in B.  
Thus, we can weite down P(B)=P(A)+P(B-A)  
Since proability is non-negative, we can know that P(B-A) ≧ 0 => P(B) ≧ P(A)
