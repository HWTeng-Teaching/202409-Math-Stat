## Prove: P(ϕ)=0

### Proof:
## Note 1
$$\ S = S \cup \emptyset \$$

## Note 2
$$\ S \cap \emptyset = \emptyset \$$, so $$\ S \$$ and $$\ \emptyset \$$ are mutually exclusive.

## Then, 
\(Note: from Axiom 3)

$$\ P(S) = P(S \cup \emptyset) = P(S) + P(\emptyset) \$$

Note: \( P(S) = 1 \) by Axiom 2, so we can substitute:

$$\ 1 = 1 + P(\emptyset) \$$

Therefore,

$$\ P(\emptyset) = 0 \$$
